# 移动端使用说明

## 1. 环境准备

### 1.1 后端启动（必须）
1. 启动 **Docker Desktop**（等待图标变绿色）
2. 打开终端，进入项目目录：
   ```powershell
   cd C:\Users\swz\WorkBuddy\2026-06-18-10-14-32\Software_Engineering
   ```
3. 启动数据库、Redis 和后端：
   ```powershell
   docker compose up db redis backend -d
   ```
4. 等待约 1 分钟，验证后端是否正常：
   ```powershell
   curl http://localhost:8000/docs
   ```
   看到 FastAPI 文档页面即表示成功。

### 1.2 前端启动（必须）
打开**新的终端窗口**，运行：
```powershell
cd C:\Users\swz\WorkBuddy\2026-06-18-10-14-32\Software_Engineering\frontend
npm run dev
```
看到 `https://localhost:5173/` 即表示成功。

### 1.3 手机访问
1. **确保手机和电脑在同一 WiFi 网络**
2. 查看电脑 IP 地址（命令行运行 `ipconfig`，找到 IPv4 地址，例如 `10.220.75.148`）
3. 手机浏览器打开：`https://你的IP:5173`
4. 出现「证书不受信任」警告 → 点 **「高级」→「继续访问」**（这是开发用的自签名证书，安全）

## 2. 功能使用说明

### 2.1 用户登录/注册
1. 打开手机浏览器，访问前端地址
2. 如果已有账号，点「登录」输入用户名和密码
3. 如果没有账号，点「注册」填写信息注册

---

### 2.2 创建 AI 模拟面试
1. 登录后，进入首页
2. 点「开始面试」或「创建面试」
3. 选择面试类型（技术面试/行为面试等）
4. 点「确认」，AI 面试官会提出第一道问题

---

### 2.3 文字回答
1. 在底部输入框输入文字
2. 点「发送」按钮（或键盘回车）
3. 等待 AI 面试官评分和反馈
4. AI 返回反馈后会自动提出下一题

---

### 2.4 语音回答（核心功能）
1. 点底部「🎤」按钮，切换到**语音输入模式**
2. 按住底部「**按住说话**」按钮，**开始录音**
3. 录音时：
   - 按钮变红色，显示录音时长（如 `00:03`）
   - 可以**上滑取消**（松手后录音不发送）
4. 松开手指，**发送录音**
5. 前端自动上传录音文件到后端
6. 等待后端转写（最多 3 分钟）：
   - 聊天区显示「转写中...」
7. 转写完成后，文本自动作为回答提交
8. AI 面试官评分并返回反馈

#### ⚠️ 语音回答注意事项
- **录音时间太短（< 1秒）**：会提示「录音时间过短，请重新录制」
- **转写超时（> 3分钟）**：会提示「转写超时，请使用文字回答」
- **麦克风权限**：手机浏览器首次使用语音会请求麦克风权限，必须点「允许」

---

### 2.5 上滑取消录音
1. 按住「按住说话」按钮开始录音
2. **不要松开**，向上滑动手指
3. 按钮变「取消」状态
4. 松开手指，录音**不会发送**
5. 可以重新录制

---

### 2.6 查看 AI 反馈
每次回答后，AI 面试官会返回：
- **评分**（0-100 分）
- **反馈意见**（回答的优点和改进建议）
- **下一题**（自动提出或等待用户点「下一题」）

---

### 2.7 查看历史面试记录
1. 点底部导航栏「**历史**」
2. 显示所有历史面试列表（按时间排序）
3. 点击任意一场面试，查看详情：
   - 面试状态（进行中/已完成）
   - 总分
   - 每题的回答、评分、反馈

---

### 2.8 查看面试报告
1. 在历史面试列表中，点「**查看报告**」
2. 报告页面显示：
   - 面试概况（总分、日期、面试类型）
   - 每题详细评分和反馈
   - 综合建议
3. 可以**下载报告**（PDF）或**复制报告内容**

## 3. 常见问题排查

### 3.1 手机打不开网页
- **原因1**：手机和电脑不在同一 WiFi
  - **解决**：连接同一 WiFi
- **原因2**：前端没启动
  - **解决**：重新运行 `npm run dev`
- **原因3**：防火墙拦截
  - **解决**：Windows 防火墙允许端口 5173 入站

### 3.2 语音录制没反应
- **原因1**：麦克风权限未授予
  - **解决**：手机浏览器设置中允许麦克风权限
- **原因2**：非 HTTPS 访问
  - **解决**：必须用 `https://` 访问（不能是 `http://`）

### 3.3 转写文本一直等待中
- **原因1**：后端 DashScope API Key 未配置或失效
  - **解决**：检查 `.env` 文件中的 `DASHSCOPE_API_KEY`
- **原因2**：录音文件过大或格式不支持
  - **解决**：缩短录音时长（建议 < 2 分钟），确保录音环境安静

### 3.4 登录/注册报 500 错误
- **原因**：后端没启动或数据库未初始化
  - **解决**：
    ```powershell
    docker compose up db redis backend -d
    # 等待 1 分钟后重试
    ```
